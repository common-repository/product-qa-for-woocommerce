== Description ==

This is an woocommerce addon for product QA which supports user interaction to give live answers,
Admin can add/edit/delete/approve all questions and answers.

== Installation ==

1. Upload this plugin folder(including all files) to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go To   Woocommerce -> Settings -> Products -> Product QA
4. For more details check Screenshots.
5. Thanks.

== Screenshots ==

1. Woocommerce Settings for Product QA.
2. Product QA listing.
3. Comments(Answers) for particular question.
4. Questions and Answers listing on front-side.
